 == Description ==
Send notification by email if Search Engine bot such as GoogleBot, Yahoo! Slurp and Msnbot has access your website

== Installation ==

1. Upload "bot-access-notification.php" to the "/wp-content/plugins/" directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. this plugin used email that registered in Settings => General. so make sure you enter valid email there.
